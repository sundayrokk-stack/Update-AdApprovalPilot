# AdApprovalPilot AI 🚀
> **The Gold Standard for Telegram Ads Compliance & Channel Auditing.**

AdApprovalPilot AI is an invite-only, production-ready Telegram bot designed for advertisers and agencies. It automates the "Destination Quality" audit process, helping users avoid rejections and long review times.

## ✨ Core Features
- **Invite-Only Access Control**: Automated approval/revocation system via @BlockSavvyMx.
- **Deep Channel Audit**: Real-time analysis of engagement, bio-compliance, and link hygiene.
- **Weighted Scoring Algorithm**: Generates a 0-100 "Ad Readiness Score".
- **Subscription Tiers**: Built-in logic for Free (limited) vs. Premium (unlimited) users.

## 🛠 Tech Stack
- **Framework**: FastAPI (Webhook mode)
- **Bot Library**: Aiogram 3.x
- **ORM**: SQLAlchemy (Async) with PostgreSQL
- **Migrations**: Alembic
- **Deployment**: Render-optimized
